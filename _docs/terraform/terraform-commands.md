Initialize terraform in a directory
```bash
terraform init
```

Apply a configuration
```bash
terraform apply
```

Destroy an infrastructure
```bash
terraform destroy
```

Show help
```bash
terraform help
```

Inspect state
```bash
terraform show
```

Validate configuration
```bash
terraform validate
```
